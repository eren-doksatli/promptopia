# Promptopia

This project is a prompt-sharing application developed using Next.js and MongoDB. Users can log in using their Google accounts to seamlessly manage their prompts. The application allows users to:

- Add new prompts
- Edit existing prompts
- Delete prompts they no longer need
- Search for relevant prompts using the integrated search functionality

This project demonstrates the capabilities of Next.js for building dynamic applications and showcases the use of MongoDB for managing data efficiently.

## Features

- User authentication via Google
- Responsive UI for prompt management
- Search functionality for easy access to prompts

## Tech Stack

- React DOM
- Mongoose
- Next
- Next-Auth
- Monodb
- Bcrypt
- Tailwindcss
